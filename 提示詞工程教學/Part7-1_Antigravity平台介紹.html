Antigravity 四大特色

1. 透明的工作流程

AI 不只給你答案，還會像員工一樣報告：

  • 提交任務清單
  • 說明執行計畫
  • 展示測試截圖

2. 雙模式操作

根據專案大小自動切換：

  • 小任務：像 IDE 一樣直接編輯
  • 大專案：管理者介面，監控多個 AI 並行

3. 智能修正

AI 完成 80% 後卡住？

  • 你可以部分批改
  • AI 會讀懂你的意圖
  • 自動修正，無需重來

4. 記憶學習

AI 會累積經驗：

  • 記住解決問題的步驟
  • 保存常用程式碼片段
  • 下次遇到類似問題更快

Step 1：註冊與設定

註冊步驟

  1. 前往 antigravity.dev
  2. 點擊「Get Started」或「Sign Up」
  3. 使用 Google 帳號登入
  4. 同意服務條款
  5. 完成！進入主畫面

免費方案包含

  • 無限制專案數量
  • 基本部署功能
  • 社群支援

建議使用環境

  • Chrome 瀏覽器（最佳）
  • 穩定的網路連線
  • 螢幕解析度 1920x1080 以上

Step 2：建立第一個專案

建立專案步驟

  1. 點擊「New Project」按鈕
  2. 輸入專案名稱：學生作品評量系統
  3. 選擇專案類型：Web Application
  4. 點擊「Create」建立

介面說明

  • 左側：檔案瀏覽器（查看專案檔案）
  • 中間：程式碼編輯區（可手動修改）
  • 右側：即時預覽（看到網站效果）
  • 底部：對話視窗（輸入指令給 AI）

Step 3：給 AI 第一個指令

在對話視窗輸入
請幫我建立一個簡單的「學生作品評量網站」首頁：

1. 標題：「學生作品評量系統」
2. 學生姓名輸入框
3. 作品內容文字區（可輸入多行）
4. 提交按鈕
5. 使用現代風格設計，紫色主題色

請使用 React 和 Tailwind CSS。

AI 會做什麼

  1. 分析你的需求
  2. 建立檔案結構
  3. 撰寫 React 組件
  4. 套用 Tailwind 樣式
  5. 在預覽區顯示結果
