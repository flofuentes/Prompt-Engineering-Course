Complete Implementation & Deployment

系統功能概述

本單元將完整實作一個學生作品評量網站，整合前端介面、Google Apps Script 後端、AI 自動評分功能。學生可透過網頁上傳作品，系統自動分析並生成評語，教師可在後台查看所有記錄。

核心功能

作品上傳

學生填寫姓名、作品描述，並可上傳 PDF/圖片等附件

AI 自動評分

整合 Gemini 或 OpenAI API，自動生成四字評語與詳細回饋

評語查詢

學生可輸入姓名查詢歷史評語，支援語音朗讀功能

教師後台

密碼保護的管理介面，可查看試算表與系統狀態

部署方式選擇

根據您的需求和技術背景，提供兩種部署方案。兩種方案都是免費的，差別在於前端網頁的託管位置。

方案 A：Google 全家桶 完全免費

前端部署於 Google Apps Script

  • 前端 HTML 直接部署在 GAS
  • 不需要額外帳號或平台
  • 適合只用 Google 服務的教師
  • 網址較長（script.google.com/...）
  • 載入速度較慢

方案 B：Netlify 部署 推薦

前端部署於 Netlify 免費空間

  • 前端部署在 Netlify（免費）
  • 可自訂短網址（xxx.netlify.app）
  • 載入速度快、體驗佳
  • 支援拖曳部署，超簡單
  • 需要註冊 Netlify 帳號

推薦選擇

如果您是第一次部署，建議選擇 方案 B（Netlify），因為：

  • 網頁載入速度更快，學生體驗更好
  • Netlify 支援拖曳資料夾直接部署，不需要寫任何指令
  • 可以獲得好記的網址，例如：your-class.netlify.app

手動部署教學

步驟一：建立 Google 試算表

  • 建立新試算表
    前往 Google Sheets，建立新的空白試算表
  • 取得試算表 ID
    從網址中複製 ID（在 /d/ 和 /edit 之間的字串）
    例如：https://docs.google.com/spreadsheets/d/YOUR_ID_HERE/edit
  • 設定表頭（可選）
    在第一列設定：時間、姓名、作品內容、四字評語、回饋、檔案連結

步驟二：取得 AI API Key

Gemini API Key（推薦）

  • 前往 Google AI Studio
  • 點擊「Create API Key」
  • 複製產生的 API Key
  • 免費額度：每分鐘 60 次請求

OpenAI API Key

  • 前往 OpenAI Platform
  • 登入後點擊「Create new secret key」
  • 複製產生的 API Key
  • 需綁定信用卡，按使用量計費

步驟三：部署 GAS 程式

  • 建立 GAS 專案
    前往 Google Apps Script，建立新專案
  • 貼上程式碼
    使用生成器產生的 GAS 程式碼，貼入編輯器並儲存
  • 啟用 Drive API
    點擊左側「服務」→ 新增「Drive API」
  • 執行授權測試
    執行 manualAuthTest 函數，授權 Google 權限
  • 部署為網路應用程式
    點擊「部署」→「新增部署」→ 選擇「網路應用程式」
    執行身分：「我」/ 存取權：「所有人」
  • 複製部署網址
    部署完成後複製產生的網址，這就是您的 API 網址

步驟四：部署前端網頁

Netlify 部署（推薦）

  1. 前往 Netlify Drop
  2. 將下載的 HTML 檔案（或整個資料夾）拖曳到網頁上
  3. 等待幾秒鐘，部署完成！
  4. 點擊「Site configuration」可修改網站名稱

GAS 部署

  1. 在同一個 GAS 專案中，建立新的 HTML 檔案
  2. 貼上生成器產生的 HTML 程式碼
  3. 修改 doGet 函數回傳 HTML 頁面
  4. 重新部署網路應用程式

測試與驗證

功能測試清單

  • 作品提交測試
    填寫姓名和作品內容，確認可成功提交並收到 AI 評語
  • 檔案上傳測試
    上傳 PDF 或圖片，確認檔案成功存入 Google Drive
  • 評語查詢測試
    在查詢頁面輸入姓名，確認可以找到剛才的提交記錄
  • 語音朗讀測試
    點擊播放按鈕，確認評語可以正常朗讀
  • 試算表確認
    開啟 Google Sheet，確認資料正確記錄

常見問題排解

  • CORS 錯誤：確認 GAS 部署時存取權設為「所有人」
  • API 無回應：檢查 API Key 是否正確、是否有啟用服務
  • 檔案無法上傳：確認 GAS 有啟用 Drive API
  • 評語亂碼：確認程式碼編碼為 UTF-8
